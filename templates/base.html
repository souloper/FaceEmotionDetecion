It's Project Info

What?

The primary goal of this project is to create a system that can receive input video feed from any camera source and detect and categorize human expressions in real-time.

Why?

Humans are unquestionably superior when it comes to emotion perception, but can computers or any algorithm do the same? That is how we approached our research.

Which?

We used an image-pixel valued dataset for this research, which is primarily a csv dataset including the 7 distinct emotion labels and pixel values of all matching photos. The dataset is split into training and validation data in the ratio of 80:20.
